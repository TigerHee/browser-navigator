## 获取浏览器信息(use js to get brower navigator information)

- 使用方法：
```sh
  var navigatorInfo = require("navigator-info");
  navigatorInfo()
```

- 得到信息实例如下：

```sh
  {
    "device":"Mobile",
    "language":"zh-cn",
    "engine":"WebKit",
    "browser":"Safari",
    "os":"iPhone",
    "version":"11.0"
  }
```
